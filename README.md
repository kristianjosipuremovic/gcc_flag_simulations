# gcc_flag_simulations

C simulations of GCC compiler hardening flag mechanisms, written as part of 
CERIAS/Caterpillar-funded binary hardening research at Purdue University.

## Methodology
Each flag has two files:
- `learning/` — annotated learning file with notes, questions, and concepts
- `final/` — clean implementation written entirely from memory
